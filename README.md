# Snehal Darade's Professional Portfolio

A premium, high-impact developer portfolio built with modern web technologies, featuring a "Cyber-Premium" aesthetic, dynamic animations, and an ATS-friendly digital resume.

## 🚀 Live Demo
[View Live Portfolio](https://your-vercel-deployment-link.vercel.app/)

## ✨ Key Features
- **Modern UI**: Dark-themed "Cyber-Premium" design with Blue-Purple gradients.
- **Glassmorphism**: Sleek, layered interface with backdrop filters.
- **Interactive Elements**: Custom cursor glow, scroll-triggered animations, and a dynamic stats counter.
- **ATS-Friendly Resume**: A dedicated, printable single-page resume optimized for recruitment systems.
- **Responsive Design**: Fully functional on Mobile, Tablet, and Desktop.

## 🛠️ Tech Stack
- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Icons**: FontAwesome 6
- **Fonts**: Google Fonts (Outfit, Inter)
- **Deployment**: Vercel

## 📂 Project Structure
- `index.html`: Main portfolio landing page.
- `resume.html`: Single-page ATS-optimized resume.
- `css/style.css`: Custom design system and animations.
- `js/script.js`: Interactivity, scroll reveals, and form handling.
- `images/`: High-end mockups and visual assets.

## 👨‍💻 How to Run Locally
1. Clone the repository:
   ```bash
   git clone https://github.com/Snehal42/portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd portfolio
   ```
3. Open `index.html` in your browser or run a local server:
   ```bash
   npx serve .
   ```
